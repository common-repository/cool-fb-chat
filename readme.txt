== Description ==
<h4>Facebook Messenger</h4>

<strong>Cool Facebook messenger</strong> plugin enables live <strong>facebook chat</strong> support inside your wordpress website. Now you can show a <strong>facebook messenger</strong> button on any page of your website using a short-code. This <strong>messenger</strong> button will connect your website visitors/users directly to your facebook page. Your users can message you directly from the website & you will receive message inside your <strong>messenger app</strong>. You can also reply to your customers instantly by enabling <strong>facbook bot</strong> auto-reply option inside your facebook page.

<h4>Cool Facebook Chat Messenger Features</h4>
<ul>
<li>Allow your clients to instantly connect with your business support team.</li>
<li>Support your customers using live fb chat.</li>
<li>Enable facebook bot for instant reply</li>
<li>Easy to install, just add your fb page.</li>
<li>Add to any page or post using shortcode.</li>
<li>Facebook contact form.</li>
<li>Cool beautiful chat button.</li>
<li>Shortcode: [cool-fb-chat]</li>
</ul>

Check facebook messenger demo: http://www.cooltimeline.com/cool-fb-chat

== Installation ==

Download the plugin .zip file

Login in to admin Click Plugins -> Add New -> Upload

Find Cool FB Chat in plugin list and activate it.
